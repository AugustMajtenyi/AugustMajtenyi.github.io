Fantasy Football Data with an API

Objective: Review Defensive Scheme Data in 2022

Context: To demonstrate using python and connecting to an API.


171 Queries Later

You can find my code here and do this for your own league.

Warning: Flashing text may potentially trigger seizures for people with photosensitive epilepsy.


What's going on

This code let's you choose which player in the league that you would like to analyze (1 query). Then it retrives your roster in every week of the 2022 season up to present (17 queires). Lastly, it reviews your starting players' stats, calculates scheme impact, and plots the data (153 queires). Each quiery prints the information in the console by default.

Defensive Scheme Impact Charts

Initial Takeaways

It appears that no scheme is noticable better than any other. There also seems to be a lot of variability from week to week. This matches firsthand experience.

Reflection

Successes

During this project, I have shown grit and I am happy with what I was able to accomplish. Special thanks to uberfastman for the api wrapper, and to my little brother for his troubleshooting help.

Limitations

I was unable to get 2021 data. Weekly stats for each player were set to "0" (see image on left).

Additionally, I was unable to find firstdown stats and cannot calculate every defensive scheme.
The four schemes I do show make up ~85% of usage on average (see image on right).


Next Steps

In it's current state, the charts are not very effect tools. More analsyis should be done if any actionable items are to be made from this. Also, not that there is a proof of concept, it might be useful to use this api to create a live-updating scoresheet that includes defensive schemes.
